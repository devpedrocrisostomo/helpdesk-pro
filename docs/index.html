HelpDesk Pro

Sistema web de helpdesk para gestao de chamados, clientes, usuarios, atendimentos tecnicos, comentarios e controle de acesso por perfil.

Ver repositorio Como testar

O que e o sistema

O HelpDesk Pro e uma aplicacao para registrar chamados de suporte, acompanhar status, vincular clientes, atribuir tecnicos e centralizar comentarios do atendimento.

  • Login com JWT.
  • Perfis de acesso: admin, tecnico e cliente.
  • Cadastro de usuarios e clientes.
  • Gestao de tickets com prioridade, status, atribuicao e comentarios.
  • Dashboard com indicadores dos chamados.

Stack tecnica

  • Java 21
  • Jakarta EE 10
  • WildFly
  • JAX-RS
  • JPA/Hibernate
  • PostgreSQL
  • Docker e Docker Compose

Como testar localmente

Clone o repositorio, copie o arquivo de ambiente e suba os containers:

cp .env.example .env
docker compose up --build

Depois acesse http://localhost:8081. Usuario inicial de desenvolvimento: admin@helpdeskpro.local / admin123.
