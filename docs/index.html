Stack

Projeto estruturado como WAR Jakarta EE com API REST, frontend estatico embarcado e banco PostgreSQL.

  • Java 21
  • Jakarta EE
  • WildFly
  • JPA/Hibernate
  • PostgreSQL
  • JWT
  • Docker

Hospedagem

Importante: Pages nao executa API, Java, Docker ou banco de dados. Para uma demo publica funcional, use Render, Railway, Fly.io, VPS ou outro host que rode Docker Compose/PostgreSQL.
